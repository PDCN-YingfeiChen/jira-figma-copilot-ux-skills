# Transfer Guide

## 推荐发送方式

把这个包作为 **No-token Trial Package** 发给其他设计师。对方无需配置 API，只需要用 mock data 验证 Skill 是否有用。

## 给设计师的话术

```text
Hi, I’m testing a Jira × Figma × Copilot UX Skill workflow.

The goal is to help UX designers use Copilot to:
1. Extract UX requirements from Jira
2. Generate user flows
3. Generate Figma design prompts
4. Compare Jira requirements with Figma design content
5. Produce two Chinese Markdown reports:
   - Design brief for the design process
   - UX review report for PM/PO, FE/BE, and QA handoff

This trial version does not require Jira or Figma tokens. It uses mock data first, so the focus is to validate whether the Skill logic and output format are useful for designers.

Could you help try the workflow and give feedback on:
- Whether the requirement summary is accurate
- Whether the user flow is useful
- Whether the design brief can support design start
- Whether the UX review report is helpful for PM/PO, FE/BE, and QA handoff
```

## 反馈重点

- 需求提炼是否准确
- User Flow 是否符合设计习惯
- 设计过程简报是否能帮助起稿
- UX 评审报告是否能帮助 PM/PO、FE/BE、QA 协作
- 输出是否适合贴回 Jira 或给 PM / Engineering 看
